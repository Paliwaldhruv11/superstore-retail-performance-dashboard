📊 Superstore Retail Performance Dashboard (Power BI)
🚀 Overview

This project presents an interactive Power BI dashboard built using the Superstore dataset to analyze retail business performance. The dashboard transforms raw data into meaningful insights, helping users understand sales trends, profitability, and key business metrics.

🎯 Objectives
Analyze overall sales and profit performance
Identify top-performing regions and categories
Track monthly trends and growth patterns
Provide actionable insights for better decision-making
📌 Key Features
📈 Sales, Profit, Quantity, and Returns Overview
🧩 Category & Sub-category Analysis
👥 Segment-wise and Payment Mode Insights
🌍 State-wise Sales and Profit Visualization (Map)
🏆 Top 10 Performing States
🚚 Ship Mode Analysis
📅 Monthly Sales & Profit Trends (Year-over-Year)
📱 Mobile-Friendly Dashboard View
